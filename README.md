# circuit_breaker

## How to work

- install

  ```bash
  yarn
  ```

- frontend build

  ```bash
  yarn frontend:build
  ```

- frontend run

  ```bash
  yarn build:dev
  ```

- smartcontract compile

  ```bash
  yarn backend:build
  ```

- smartcontract test

  ```bash
  yarn backend:test
  ```

- smartcontract deploy

  ```bash
  yarn backend:deploy:scrollSepolia
  ```
